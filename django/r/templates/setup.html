Setup the R App

This app depends on R, which is an external library for statistical computation. Please download and install the library for your OS:

Download for Mac Download for Windows Download for Linux/Unix

After installing, please check the installation path. For Windows, it should be something like C:\Program Files\R\R-3.0.3\bin\R.exe. For Mac OS X and Linux or Unix, it's probably /usr/bin/R.

The app works as follows. When you run the r command, the data from your Splunk pipeline is saved as a CSV file, and loaded into R using read.csv. Your custom script is then executed on the input variable, the output variable is saved using write.csv, and this final CSV is loaded back into Splunk.
